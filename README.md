### update [done]
updates things. pretty simple.

### ssh
configures ssh, sshd, etc.

### samba
configures samba

### login [done]
configures login stuff, like password length, root login, etc.

### malware [done]
scans for malware and other nasty stuff.

### telnet
configures telnet.

### users
changes permissions, removes unauthorized users.
